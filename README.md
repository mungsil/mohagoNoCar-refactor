# 🚗❌ 뭐하고 No Car (서버)

**뭐하고 No Car**는 **자가용 없이**도 **편리하고 즐거운 여행**을 할 수 있도록 지원합니다. 다음과 같은 분들에게 추천합니다.

- **자가용 없이** 여행을 계획하는 분들
- **여행 도중 지역 축제에** 참여하고 싶은 분들
- **효율적인 경로로 여행지를 방문하는** 것을 선호하는 분들

---

## 🌟 주요 기능

- **대중교통 여행 계획 지원**: 지역 축제 중심의 자가용 없이도 편리하게 여행을 다닐 수 있는 코스 설계
- **지역 축제 정보**: 다양한 지역 축제의 일정과 정보를 한눈에 확인
- **지역 축제 주변 맛집/명소 정보**: 지역 축제 주변에 있는 맛집과 관광지 확인
